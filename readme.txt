Facebook-like phototagging functionality in your WordPress blog.

== Description ==

Add Facebook-like phototagging to your blog.

Select any photo in your media library, tag portions of it, and then display the image and tags in your posts.

This is still in beta, and there are some known issues.  See http://jontascher.com/phototagger for more details.
